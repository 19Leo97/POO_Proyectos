CONTROLES: 
    Click: Matar, revivir
    Espacio: Pausa, Play
    Izquierda,Derecha: Disminuir,Aumentar velocidad
    Patrones:
        0:Reset(Extinción)
        1:Aleatorio
        2-9:Demas patrones, extraidos de https://www.conwaylife.com/wiki/Main_Page
